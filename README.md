# Student Record Management System

This is a simple web-based system I developed in my second year of university as my very first project. It was built to practice core programming concepts such as CRUD operations, PHP basics, and database handling. The system provides a straightforward way to manage student records.


##  Features
- Add new student records
- View and search student details
- Update existing records
- Delete records
- Basic validation and error handling


## Tech Stack
- Frontend: HTML, CSS
- Backend:PHP 
- Database: MySQL
- Server Environment: XAMPP


##  Project Structure
- `index.php` → Main entry point
- `add_student.php` → Form for adding records
- `update_student.php` → Edit existing records
- `delete_student.php` → Remove records
- `db/connection.php` → Database connection file
